Sobre a Nex_TI

A Nex_TI é uma EdTech nascida de um Projeto Integrado Multidisciplinar (PIM III). Nosso objetivo principal é combater a Curva de Esquecimento de Ebbinghaus no processo de aprendizado de tecnologia através do uso massivo do Algoritmo SM-2 (Repetição Espaçada) acoplado a um ecossistema gamificado.

Metodologia de Ensino

Substituímos o aprendizado passivo por estímulos de Estudo Ativo. Ao invés de vídeos longos, o aluno consome conceitos atômicos através de "Flashcards" interativos, onde a performance no recall do conhecimento (Erro/Acerto) dita quando aquela informação será apresentada novamente.

Manifesto Open Source

Acreditamos que o código que molda o futuro da educação deve ser aberto. O núcleo do Nex_TI, incluindo a lógica do Anki Engine e a arquitetura em .NET 10, está disponível livremente no GitHub para auditoria, extensão e bifurcação pela comunidade.

Carreiras & Vagas

Estamos sempre em busca de talentos que queiram revolucionar a educação no Brasil. No momento, somos um projeto de extensão universitária e não possuímos vagas CLT abertas, mas contribuições via Pull Request no GitHub são muito bem-vindas!

Trilhas e Disciplinas

Nossa plataforma é modular. Oferecemos pacotes curriculares focados em Engenharia de Software, Lógica de Programação, Cibersegurança e Arquitetura de Nuvem. Cada trilha é dividida em missões progressivas que dão XP ao serem concluídas.

Simulado ENADE

Módulo exclusivo de preparação para o ENADE, contendo questões reais de provas anteriores de Análise e Desenvolvimento de Sistemas formatadas no modelo de Flashcard. Responda, verifique o verso para explicação detalhada e ganhe pontuação dobrada.

Comunidade Devs

A seção de comunidade visa conectar alunos em diferentes níveis (Júnior, Pleno e Sênior) baseados no XP da plataforma. É um espaço assíncrono para tirar dúvidas de código e trocar experiências sobre carreira (Funcionalidade prevista para V2).

Loja de Bônus (Coins)

As Moedas Virtuais (Coins) que você minera estudando diariamente servem para desbloquear Flashcards ocultos de disciplinas avançadas ou "Easter Eggs" sobre cultura hacker e curiosidades de TI. Quanto mais você estuda, mais acesso você tem.

Central de Ajuda (FAQ)

P: Como recomeço o meu progresso?
R: No Dashboard, abra o menu superior direito e clique em "Formatar Sistema". ISSO APAGARÁ TODO SEU XP LOCAL.

P: Como o algoritmo decide quando vou ver a carta de novo?
R: Usamos o SM-2 modificado. Se você errar, verá a carta logo em seguida. Se acertar, ela vai para a caixa de revisão de amanhã. Acertos consecutivos multiplicam o tempo de distância.

Termos de Uso

Ao acessar o Nex_TI, você compreende que trata-se de um sistema de avaliação e extensão acadêmica. É proibido o uso de automação ou bots para inflar XP e farmar Moedas. Contas identificadas com comportamento malicioso serão expurgadas do servidor SQL.

Aviso de Privacidade (LGPD)

A Nex_TI atua em conformidade com a LGPD. Não capturamos cookies de rastreamento de terceiros. Seu E-mail e Username são mantidos em banco de dados isolado e suas senhas são armazenadas através de hash unidirecional (BCrypt). Você tem total controle para solicitar a deleção do seu perfil ("Esquecimento").

Contato do Desenvolvedor

Para suporte técnico, bugs na API ou reportar vulnerabilidades de segurança (Bug Bounty), abra uma "Issue" no repositório oficial do GitHub ou comunique-se com a equipe do PIM III através dos canais da UNIP.
